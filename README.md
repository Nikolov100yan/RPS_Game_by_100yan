# Rock–Paper–Scissors (Console Game)

A console-based Rock–Paper–Scissors game written in Python.
The game includes input validation, persistent score tracking,
and modular function-based design.

## Features
- Play Rock, Paper, Scissors against the computer
- Persistent score tracking (wins, ties, losses)
- Input validation
- Clean console interface

## How to Run
1. Make sure Python 3 is installed
2. Run the script:
   python rps.py

## Controls
- r → Rock
- p → Paper
- s → Scissors
- q → Quit

## Purpose
This project was created to practice Python fundamentals,
including functions, loops, dictionaries, and program structure.
